# Fake News Detection using TensorFlow

This project detects whether a news article is FAKE or REAL using Deep Learning.

## Technologies Used
- Python
- TensorFlow
- Pandas
- NumPy
- Scikit-learn
- NLP
- LSTM

## Project Workflow
1. Import libraries
2. Load dataset
3. Clean text data
4. Label encoding
5. Tokenization
6. Padding sequences
7. Build LSTM model
8. Train model
9. Evaluate model
10. Predict new news article

## Model Performance
- Accuracy: 77%
- Precision: 81%
- Recall: 69%
- F1-score: 75%
